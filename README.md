# Sales-Analysis-Excel-Dashboard

The Sales Analysis Dashboard is an interactive business intelligence dashboard developed in Microsoft Excel to help management monitor sales performance, customer behavior, product performance, and regional trends.

The dashboard consolidates sales data into meaningful Key Performance Indicators (KPIs), charts, and slicers, enabling decision-makers to quickly identify sales trends, top-performing products, customer spending patterns, and revenue distribution across different categories and occasions.

This dashboard eliminates manual reporting and provides a single view of business performance for faster decision-making.

🔧 Tools Used

Microsoft Excel
Power Query
Pivot Tables
Charts & Slicers

🧠 Key Learnings
Data cleaning and preparation
Merging datasets using Power Query
Analytical thinking with pivot tables
Building interactive dashboards in Excel

📈 Business Problems Solved
1. Revenue by Month
2. Revenue by Category
3. Top 10 Cities by Orders
4. Top 5 Products by Revenue
5. Revenue by Order Time
